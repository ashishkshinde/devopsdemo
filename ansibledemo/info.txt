To install Ansible on RHEL Host :
    Ansible is a python product, we can use python package manager pip to install it.
    
sudo pip3.11 install ansible

To test the connectivity between master and slave device :

ansible-playbook -i <host-ip> -e ansible_user=<user_name> ansible_password=<pwd> -m ping

Below command is used to manage config of a specific host :

ansible-playbook -i <host-ip> -e ansible_user=<user_name> ansible_password=<pwd>
